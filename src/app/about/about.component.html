🌱 About GreenScan

This project is developed by **MCA students** as a part of our academic learning experience. The **GreenScan** website allows users to scan a plant's QR code and instantly retrieve detailed information stored in the database.

🌍 Features of This Project

  • 📌 Scan plant QR codes and get instant details.
  • 📌 All plant information is securely stored in a database.
  • 📌 Website is **fully hosted online** for easy access.
  • 📌 Simple and user-friendly interface.

📢 Why We Built This?

We created this project to **digitally enhance plant identification** and contribute towards a smarter, more informative way of learning about plants.

🚀 Technologies Used

🔹 Angular 🔹 Firebase Hosting 🔹 Database Integration 🔹 QR Code Scanning
